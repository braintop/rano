# עורך טקסט עשיר למאמרים ✨

## התקנה הושלמה! 🎉

הוספתי לפרויקט עורך טקסט עשיר (Rich Text Editor) מלא למאמרים, בדיוק כמו שיש בפרויקט השני.

## איך להתחיל? 🚀

### 1. כניסה לדף הניהול
```
http://localhost:8080/0522577194/admin
```

### 2. לחץ על "מאמרים" בתפריט העליון

### 3. צור מאמר חדש!

## מה יש בעורך? 🎨

### עיצוב טקסט
- **מודגש** | *נטוי* | <u>קו תחתון</u> | ~~קו חוצה~~
- צבע טקסט 🎨
- צבע רקע (Highlight) 🖍️

### מבנה
- כותרות (H1, H2, H3)
- רשימות עם תבליטים ●
- רשימות ממוספרות 1️⃣
- בלוק קוד עם Syntax Highlighting 💻
- קישורים 🔗

### מדיה
- **YouTube** 🎥 (כפתור אדום)
- **Vimeo** 🎬 (כפתור כחול)

### נוסף
- יישור (ימין/מרכז/שמאל)
- Undo/Redo ↩️↪️
- תמיכה מלאה בעברית 🇮🇱

## דוגמה מהירה

1. **צור מאמר חדש**
2. **הזן Slug**: `cyber-security-2024`
3. **בחר סטטוס**: פורסם
4. **מלא תוכן** בעברית ו/או אנגלית
5. **שמור** ✅

המאמר יופיע ב:
```
http://localhost:8080/articles/cyber-security-2024
```

## טיפים 💡

- **בלוק קוד**: אוטומטית LTR (שמאל לימין)
- **וידאו**: אפשר להדביק URL או קוד iframe
- **Slug**: באנגלית עם מקפים (my-article-title)
