# כפתורי שיתוף WhatsApp למאמרים 🟢

## מה הוסף? ✅

הוספתי כפתורי WhatsApp לשיתוף מאמרים ב-**3 מקומות**:

---

## 1️⃣ **AdminArticles** - דף ניהול מאמרים (Back Office)
**מיקום:** `/0522577194/admin/articles`

### מה יש:
- ✅ כפתור WhatsApp ירוק ליד כל מאמר
- ✅ ממוקם בין כפתור "ערוך" לכפתור "מחק"
- ✅ שולח: כותרת המאמר + לינק למאמר

### איך זה נראה:
```
[ערוך] [WhatsApp 🟢] [מחק 🗑️]
```

### מה נשלח בוואטסאפ:
```
אחריות דירקטורים: פסק דין חדש והשלכות על ביטוח D and O בישראל

https://www.ranw.tech/articles/directors-liability-court-ruling-do-insurance-israel
```

---

## 2️⃣ **BlogArticles** - סקשן המאמרים בעמוד הבית
**מיקום:** `/#blog-articles`

### מה יש:
- ✅ כפתור WhatsApp ירוק קטן בפינה הימנית העליונה
- ✅ מופיע רק ב-**hover** (ריחוף עם העכבר)
- ✅ לא מפריע לעיצוב הכרטיס
- ✅ שולח: כותרת המאמר + לינק למאמר

### איך זה נראה:
כשמרחפים על כרטיס המאמר → כפתור ירוק קטן מופיע למעלה בצד ימין 🟢

---

## 3️⃣ **ArticlePage** - דף המאמר עצמו
**מיקום:** `/articles/:slug`

### מה יש:
- ✅ כפתור "שתף" גדול בפינה הימנית העליונה
- ✅ ליד הכותרת של המאמר
- ✅ צבע ירוק של WhatsApp
- ✅ שולח: כותרת המאמר + URL הנוכחי

### איך זה נראה:
```
┌─────────────────────────────────────────┐
│ כותרת המאמר                    [שתף 🟢] │
│ תאריך פרסום                             │
│ זמן קריאה                               │
└─────────────────────────────────────────┘
```

---

## 🔧 איך זה עובד?

### קוד WhatsApp Share:
```typescript
const url = `https://www.ranw.tech/articles/${slug}`;
const text = encodeURIComponent(`${title}\n\n${url}`);
window.open(`https://wa.me/?text=${text}`, '_blank');
```

### מה זה עושה:
1. **יוצר URL** - לינק מלא למאמר
2. **מקודד טקסט** - כותרת + שורה ריקה + לינק
3. **פותח WhatsApp** - בטאב חדש עם הטקסט מוכן לשליחה

## 🎨 עיצוב

### צבעים:
- **רקע:** `bg-green-500` (ירוק WhatsApp)
- **Hover:** `hover:bg-green-600` (ירוק כהה יותר)
- **טקסט:** `text-white` (לבן)

### אייקון:
- **lucide-react:** `Share2`
- **גודל Admin:** `h-4 w-4` (קטן)
- **גודל Frontend:** `h-5 w-5` (בינוני)

## 📱 תצוגה במובייל

הכפתורים מותאמים לנייד:
- **AdminArticles:** הכפתור נשאר תמיד גלוי
- **BlogArticles:** הכפתור גלוי תמיד (לא רק ב-hover)
- **ArticlePage:** כפתור "שתף" גדול ובולט

## ✅ בדיקה מהירה

